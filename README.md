# ✦ SocialForge — AI Social Media Marketing Agent

AI-powered social media management tool built with React + Vite, powered by Google Gemini 2.0 Flash.

## Features

- **Brand & Goals** — Define your product, audience, tone, and objectives
- **Content Calendar** — Plan posts manually or auto-generate monthly plans with AI
- **Create Content** — Generate platform-optimized social media copy in 3 variations
- **Image Studio** — Upload product images and manage AI image generation prompts
- **Campaign Manager** — Create long-running campaigns with AI month-by-month execution plans
- **Performance Analytics** — Log metrics, visualize engagement, get AI analysis
- **Review & Adapt** — Paste competitor campaigns for AI review and brand adaptation

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) 18+ installed
- A free [Gemini API key](https://aistudio.google.com/apikey)

### Local Development
```bash
npm install
npm run dev
```
Open http://localhost:5173/socialforge/ and add your Gemini API key in Settings.
